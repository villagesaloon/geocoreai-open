API 说明

接口地址： GET /api/trends/heat?keyword=xxx

返回数据：

{
  "keyword": "ChatGPT",
  "avgHeat": 79,      // 过去12个月平均热度 (0-100)
  "recentHeat": 67,   // 最近一周热度 (0-100)
  "relatedQueries": ["chatgpt ai", "free chatgpt", ...],
  "success": true
}

限流： 每个用户每分钟最多 10 次请求
